## ifup

激活指定的网络接口

### 补充说明

**ifup命令** 用于激活指定的网络接口。

### 语法

``` plain
ifup (参数)
```
### 参数

网络接口：要激活的网络接口。

### 实例

``` plain
ifup eth0   #激活eth0
```
## ifdown

禁用指定的网络接口

### 补充说明

**ifdown命令** 用于禁用指定的网络接口。

### 语法

``` plain
ifdown (参数)
```
### 参数

网络接口：要禁用的网络接口。

### 实例

``` plain
ifdown eth0  #禁用eth0
```
